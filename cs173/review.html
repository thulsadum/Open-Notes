  4. Sets:

    4. Inclusion: Definition: A ⊆ B iff {∀x, x ∈ A → x ∈ B}
    5. Set operations
      1. Intersection: A ∩B = {S | S ∈ A ∧S ∈ B}
        Disjoint: A ∩B = ∅
      2. Union: A ∪B = {x:x ∈ A ∨x ∈ B}
      3. Difference: A−B = {x:x ∈ A ∧x ∉ B}
      4. Complement ―A = {x: x ∉ A ∧x ∈ U}
      5. Cartesian product: A ×B = {(x, y) | x ∈ A ∧ y ∈ B}
    6. DeMorgan’s Law: ―A ∪B = ―A ∩―B
    7. Inclusion-Exclusion Principle: |A ∪B| = |A| + |B| − |A ∩B|

  5. Relations
    1. Relation: A relation R on a set A is a subset of A ×A, i.e. R is a set of ordered pairs of elements from A.
    2. Reflexive: R is reﬂexive if xRx for all x ∈ A
    3. Irreflexive R is irreﬂexive if x not R x for all x ∈ A.
    4. Symmetric: ∀x, y ∈ A, xRy → yRx
    5. Antisymmetric: ∀x, y ∈ A,(xRy ∧yRx) → x = y
    6. Transitive: ∀a, b, c ∈ A, if aRb and bRc, then aRc
    7. Types of relations:
      1. Partial order: a relation that is reﬂexive, antisymmetric, and transitive.
      2. Linear order (total order): partial order in which every pair of elements are comparable. That is, for every pair x, y, x ≥ y or y ≥ x.
      3. Strict partial order: a relation that is irreﬂexive, antisymmetric, and transitive.
      4. Equivalence relation: a relation that is reﬂexive, symmetric, and transitive.
  6. Functions
    1. Function: For each input value, a function must provide one and only one output value.
    2. Image: f(A) = f(x) : x ∈ A
    3. Onto(surjective): ∀y ∈ B, ∃x ∈ A, f(x) = y
    4. One-to-One(injective): ∀x, y ∈ A, f (x) = f (y) → x = y
    5. Bijection(one-to-one correspondence): Onto and One-to-One
    6. Pigeonhole principle: Suppose you have n objects and assign k labels to these objects. If n > k, then two objects must get the same label.

  8. Graphs
    1. Simple Graph: A graph with neither multiple edges nor loop edges
    2. An isomorphism from G1 to G2 is a bijection f : V1 → V2 such that vertices a and b are joined by an edge iff f (a) and f (b) are joined by an edge.
    3. Complete graphs: Kn is a graph with n vertices in which every vertex is connected to every other vertex. ([(n(n−1))/2]) edges
    4. Cycles: Suppose that we have n vertices named v1 , . . . , vn , where n ≥ 3. Then the cycle Cn is the graph with these vertices and edges connecting vi to vi+1 , plus an additional edge from vn to v1. n edges
    5. Wheels: The wheel Wn is just like the cycle Cn except that it has an additional central “hub” node which is connected to all the others. Notice that Wn has n + 1 nodes (not n nodes). It has 2n edges.
    6. Walk: a walk of length k from vertex a to vertex b is a ﬁnite sequence of vertices a = v1 , v2 , . . . , vn = b and a ﬁnite sequence of edges e1 , e2 , . . . , en−1 in which ei connects vi and vi+1 , for all i
    7. Path: A path is a walk (necessarily an open walk) in which no vertex is used more than once.
    8. Acyclic: Doesn't contain any cycle
    9. Cut edge: One edge is keeping the graph as one connected component
    10. Euler Circuit: An Euler circuit of a graph G is a closed walk that uses each edge of the graph exactly once.
    11. Bipartite: A graph G = (V, E) is bipartite if we can split V into two non-overlapping subsets V1 and V2 such that every edge in G connects an element of V1 with an element of V2 .
    12. HyperCube:
      1. Q0 is a single vertex with no edges
      2. Qn consists of two copies of Qn−1 with edges joining corresponding vertices.
