  2. $\mathbb{N}$ in this class starts at 0
  3. A set X is countably infinite iff $∃$ a bijection $f:\mathbb{N} → X$

  1. String(w) = sequence of characted in Σ
  2. $w:\{ci∈Σ |0\leq i\leq n\}$
  3. $|w|= n ≡$ length of the string
  4. $|w| = 0 → w = ε$

  5. Substring subsequence of characters in w
  6. Concatination: $w1 ⋅ w2$
  7. Reverse: $wr$

2 Regular Languages

2.1 Deterministic Finite Automotas

  1. Finite state machine (M)
  2. Takes a string of inputs
  3. 2 types of states
    1. Accept
    2. Deny
  4. There is 1 start state
  5. The set of all strings accepted by language of M or L(A)
  6. Formal Definition
    1. a Language A $∈ Σ*$ is called regular iff there exists a DFA ,M, s.t. $L(M) = A$
    2. A DFA is a 5 tuple $M = (Q,Σ, δ, q0, F)$
      1. Q is a finte set of states
      2. Σ is a finite alphabet
      3. $δ: Q×Σ → Q$ is the transition function
      4. $q0 ∈ Q$ is the inital state
      5. $F ⊂ Q$ is the set of accept states
    3. L(M) ≡ language of all accepted strings
  7. Closure properties/Regular Operations on languages
    1. A1 and A2 are regular
    2. Union: $A1 ∪ A2 = A3$
    3. Concatenate: $A1 A2 = A3$
    4. Star: $A1* = A3$

2.2 Non-Deterministic Finite Automotas (NFA)

  1. Formal Definition
    1. $M = (Q,Σ,δ,q0,F)$
      1. Q = finite set of states
      2. Σ is a finate alphabet
      3. $δ = Q × Σε → Pow(Q)$
        1. $Σε = Σ ∪ {ε}$
      4. $q0$ = start state
      5. $F ⊂ Q$
    2. NFA accepts w If we can write $w = {y1 y2 … yn yi ∈ Σε}$ s.t. there exists a sequence of states path $R = {r0,r1,…}$
      1. r0 = q0
      2. $ri+1∈δ(ri,yi+1)$ for $i=0,1…m-1$
      3. $rm ∈ F$
  2. Useful Lemma: For all NFA, M, there exists an DFA ,N, s.t. L(m) = L(n)

2.3 Regular Expressions

  1. Def: R is a regex over a fixed alphabet iff one of the following is true:
    1. $R=a∈Σ$
    2. $R=σ$
    3. $R=∅$
    4. $R=R1∪ R2$, given $R1$ and $R2$ are regex
    5. $R = R1 R2$, given $R1 ∧ R2$ are regex
    6. $R = R*$
  2. Order of operations
    1. star
    2. concatination
    3. union
  3. Identities
    1. $a ∅ = ∅$
    2. $a σ = σ$
    3. $∅* = {σ}$

2.4 Generalized NFA (GNFA)

  1. Definition
    1. Q = set of all states
    2. $Q0 = Q - {qstart, qaccept}$
    3. The start state has out edges to every $q∈ Q - {qstart}$, and no in edges
    4. The accept state has inedges from every $q∈ Q - {qaccept}$, and no outedges
    5. An edege exists from every $q1∈ Q0$ to every $q2∈ Q0$ even if $q1=q2$
    6. Every edge is labed with a regex
  2. Useful lemma: Any NFA can be written as a GNFA
  3. lemma: Given a GNFA, M, with 2 states, the regex between the 2 states describes the language of M

If A is regular, then $∃ p ∈ \mathbb{N}$ s.t. $∀ s ∈ A$ for which $|s| >= p, s$ can be written as $xyz$ and satisfy the following conidition:

  1. $∀ i >= 0, x yi z ∈ A$
  2. $|y|>0$ i.e. $y ≠ ε$
  3. $|xy| \leq p$

2.9 Right Invariant Equivilance

Def: An equivilance relation is called right invariant or concatenation invariant iff $x∼ y \implies ∀ w ∈ Σ* xw∼ yw$

2.10 Myhill-Nerod Theorem

  • Claim: The following statments are equivalent
    1. A is a regular language
    2. $∃$ a right invariant equivilance relation that has a finite index, and $A=$ union of some of the equivilance classes
    3. $\stackrel{A}{∼}$ is of finite index
  • Proof: $1→ 2$
    1. Let M be any DFA, $M=(Q,Σ,δ,q0,F)$ s.t. $A=L(M)$
    2. Let $\stackrel{A}{∼}$ be define as in Right Invariant Equivilance
    3. The number of equivilance classes is $\leq |Q|$
    4. A is then the union of classes that correspond to F $\square$
  • Proof: $2→ 3$
    1. Show that the partition of $Σ*$ produced by a right invariant is a refinement of the parition induced by $\stackrel{A}{∼}$
    2. Since $∼$ is right invariant, $∀ z∈ Σ*,xz∼ yz$ which implies that $xz∈ A$ iff $yz ∈ A$ which by definition implies $x\stackrel{A}{∼} y$
  • Proof: $3→ 1$
    1. Construct a DFA using $\stackrel{A}{∼}$
    2. Let $Q ≡$ set of equivilance classes of $\stackrel{A}{∼}$
    3. Let $[x]∈ Σ*, [x]∈ Q$, denote the equivilance class that x belongs to
    4. Let $δ([x],a)≡ [xa]$, by def of $\stackrel{A}{∼}$
    5. Let $q0 ≡ [ε]$ and $F≡{[x] : x∈ A}$

3 Context Free Langagues

3.1 Formal Definition:

  1. $(V,Σ, R, S)$
    1. $V =$ Finite set of variables or "non-terminals"
    2. Σ = finite set of terminals
      1. $Σ ∩ V = ∅$
      2. Convention: Variables are uppercase, symbols are lowercase
    3. R = finite set of rules or "subsitution rules" or "productions" 1.Rules: examples
      1. $A → aaBc | a$
        1. This means the for an A you can replace it with aaBc or a
      2. $A ⇒ OA1 ⇒ 00A11 ⇒ 001A011 ⇒ 001011$
    4. S is the start variable
  2. $L(G) = \{w∈ Σ* | S \productionstar w\}$
  3. Notation:
    1. Variables: A,B,C…
    2. Terminal: a,b,c,…0,1,$ε$
    3. $U \productionstar V$ is defined as $∃$ sequence $U1..Uk$ , s.t. $U ⇒ U1 ⇒ U2 ⇒ … ⇒ Uk → V$

3.2 Chomsky Normal form (CNF)

  • All rules have the form
    • $A→ BC$, where $B,C$ cannot be $S$
    • $A→ a$
    • if $A→ ε$ then $A = S$
  • Lemma: Any CFG can be written in CNF

3.3 Deterministic Push Down Automotas

  • $M = (Q,Σ, Γ, δ, q0, F)$
    • $δ: Q × Σε × Γε → Q × Γε$

3.4 Non-Deterministic Push Down Automotas (PDA)

  • $M = (Q,Σ, Γ, δ, q0, F)$
    • $δ: Q × Σε × Γε → Pow(Q × Γε)$

3.5 Relating PDA to CFL

  • A language is context free iff $∃$ a PDA that recognizes it
    • Lemma: If A is CF, then $∃$ a PDA,$M$, s.t. $A = L(M)$
    • Lemma: $∀$ PDA, $M$, $∃$ CFL, $G$, s.t. $L(G)=L(M)$
      • Proof Idea: Make a conical PDA (while preserving acceptance) as follows
        1. 1 accept states
        2. Stack is empty when accepting
        3. Every transition either push or pops but not both

3.6 Pumping Lemma for CFL's

Theorem: If A is a CFL, then $∃ p \geq 0, p∈\mathbb{Z} s.t. ∀ s∈ A : |S| \geq p \implies ∃$ a paritition $s=uvxyz$ that satisfy the follow conditions

  1. $∀ i \geq 0, uvixyiz ∈ A$
  2. $|vy| > 0$
  3. $|vxy| \leq p$

Proof:

  1. Let G be a CFG s.t. $A=L(G)$
  2. Let $b$ be the max length of the right side of a rule $∈ R$ and assume $b \geq 2$
    1. If $b < 2$ the language must be finite thus the pumping lemma is trivially true
  3. Consider the derivation tree if the tree height is $h$, then the length of the generated string,s, $\leq bh$
  4. Let p = b|V|+1, where $V =$ set of variables
  5. Observe that for any $s∈ A$ that $|S| \geq p \implies h \geq |v|+1$
  6. Choose the 'smallest' derivation tree by height for s
  7. The longest path has length $|v|+1$ and vists $|v|+2$ variables
  8. Note that $|v|<$ variables visted, thus by the Pidgen Hole Principle there must be at least 1 variable repeated
  9. Thus There is a cycle in the production process strings which can then be repeated an indefiniate amount of times in the form $uvixyiz \square$

3.7.1 Theorems for Closure

Let $Gi = (Vi, Σi, Ri, Si)$ for $i=1,2$ and $Ai = L(Gi)$
Without loss of generality, assume $V1 ∩ V2 = ∅ ∧ S3$

  • Theorem: If A1 and A2 are CFL's, then A1 ∪ A2 is a CFL
    Proof:
    1. Let $Gi = (Vi, Σi, Ri, Si)$ for $i=1,2$ and $Ai = L(Gi)$
    2. Without loss of generality, assume $V1 ∩ V2 = ∅ ∧ S3 ∉ V1 ∪ V2$
    3. Construct $G3 = (V1 ∪ V2 ∪ \{S3\}, Σ1 ∪ Σ2, R3, S3)$ with $R3= R1 ∪ R2 ∪ \{S3 → S1|S2\}. \square$
  • Theorem: If A1 and A2 are CFL's then A1 ⋅ A2 is a CFL
    Proof:
    1. $∉ V1 ∪ V2$
    2. Construct $G3 = (V1 ∪ V2 ∪ \{S3\}, Σ1 ∪ Σ2, R1 ∪ R2 ∪ \{S3 → S1 S2\}, S3)$
  • Theorem: If A1 and A2 are CFL's then A1* is a CFL
    Proof: Construct $G3 = (V1 ∪ \{S3\}, Σ1, R1 ∪ \{S2 → S1 S2 | ε \})$

let the grammar contain r nonterminal symbols R1 ... Rr.
This grammar contains the subset Rs which is the set of start symbols.
let P[n,n,r] be an array of booleans. Initialize all elements of P to false.
"""

for each i = 1 to n:
    for each unit production Rj -> ai:

            for production in A:
                if P[j][k][B] and P[j+k][i-k][C]:
                    P[j][i][A] = True
if any P[1][n]: #x is iterated over the set s, where s are all the indices for Rs)
    print 'S is member of language'

5 Turing Machines

5.1 Formal Def

A Turing Machine is a 7-tuple $(Q,Σ,Γ,δ,q0, qaccept,q reject)$ with $QΣ,Γ≡$ non empty sets

  • Q is set of states
  • Σ is the input alphabet, which does not contain the blank symbol _
  • Γ is the tape alphabet, in which $_∈Γ$ and $Σ ⊂Γ$
  • δ : $Q×Γ→ Q×Γ× \{L,R\}$ is the transition function
  • $q0∈ Q$ is the start state
  • $qaccept∈ Q$ is the accept state
  • $qreject∈ Q$ is the reject state
  • $qaccept≠ qreject$

5.2 Configurations

A configuration of the turing machine $≡ ci =(qi,pi,ti)$, where $qi ∈ Q,pi$ is the head pos, and $ti∈ Γ*$ is the tape contents.

Notice that configurations are unique, and from them given the next input symbol one can determine the next configuration. i.e. \begin{equation} δ(qi,γi):(ci,γi) \mapsto (qi+1,pi+1,Di+1) \mapsto ci+1 \end{equation} for $Di+1∈ \{L,R\}$

5.2.1 Halting Configs

If either of the following type of configs is encountered, the turning machine halts and returns 'accept' or 'reject' respectively.

  • Accept Config
    \begin{equation} caccept \iff qi = qaccept \end{equation}
  • Reject Config
    \begin{equation} creject \iff qi = qreject \end{equation}

5.7 Undecidability

5.7.1 Halting Problem

  1. Theorem: $ATM$ is not Turing Decidable
  2. Proof by Contradiction:
    1. Suppose $ATM$ were decidable.
    2. Let H be a TM that decides $ATM$
      1. $H(\langle M, w \rangle) =$
        1. accept if $w ∈ L(M)$
        2. reject if $w ∉ L(M)$
    3. Construct a TM, D, which uses H and give the opposite result
      1. $D ≡$ on input $\langle M \rangle$, in which M is a T.M
        1. Run H on input $\langle M, \langle M \rangle \rangle$
        2. Return the opposite of what H outputs
    4. Therefore $D(\langle M \rangle):$
      1. accept if M rejects $\langle M\rangle$
      2. reject if M accepts $\langle M\rangle$
    5. Note that this implies $D(\langle D \rangle):$
      1. accept if D rejects $\langle D\rangle$
      2. reject if D accepts $\langle D\rangle$
    6. Notice that this is a contradiction, thus H and D cannot exist

5.8 Reduction

Using a language that is known to be undecidable prove that another language is not undecidable.

5.8.1 Map Reductions

  • Computable Functions
    A function $f:Σ* → Σ*$ is called computable iff $∃$ a TM, M, s.t. M halts $∀ w∈ Σ*$, and after halting, $f(w)$ appears alone on the tape.
  • Mapping reducible
    A language A is called mapping reducible to language B, written $A \leqM B$ iff $∃$ a computable function $f: Σ* → Σ*$ s.t. $∀ w ∈ Σ* , w ∈ A \iff f(w) ∈ B$
  • REGULARTM

5.8.2 Rice's Theorem:

  1. If P is a set of TM's with a property that satisfies:
    1. $∀$ TM's $M$ and $M2$ s.t. $L(M1) = L(M2)$, $\langle M1 \rangle ∈ P$ iff $\langle M2 \rangle ∈ P$
    2. $∃$ TM M1 and M2 for which $\langle M1 \rangle ∈ P$ and $\langle M2 \rangle ∉ P$
  2. Then the language of P is not Turing Decidable
