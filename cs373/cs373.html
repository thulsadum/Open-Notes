1.1 Starting off

  1. Alphabet(Σ) = finite non empty set
  2. \mathbb{N} in this class starts at 0
  3. A set X is countably infinite iff $∃$ a bijection $f:\mathbb{N} → X$

1.2 Strings

  1. String(w) = sequence of characted in Σ
  2. w:{ci ∈ Σ | 0 <= i <= n}
  3. |w| = n = length of the string
  4. |w| = 0 → w = ε
    1. Careful σ ≠ ∅
  5. Substring subsequence of characters in w
  6. Concatination: w1 ⋅ w2
  7. Reverse: wr
  8. Palindrum: w = wr

1.3 Language

  1. Language(L) = set of strings
  2. Σn = {w : |w| = n}
  3. Σ0 = { ε }
  4. Σ* = ∪ni=0 Σi, Language of all strings

2 Regular Languages

2.1 Deterministic Finite Automotas

  1. Finite state machine (M)
  2. Takes a string of inputs
  3. 2 types of states
    1. Accept
    2. Deny
  4. There is 1 start state
  5. The set of all strings accepted by language of M or L(A)
  6. Formal Definition
    1. a Language A $∈ Σ*$ is called regular iff there exists a DFA ,M, s.t. $L(M) = A$
    2. \A DFA is a 5 tuple $ M = (Q,Σ, δ, q0, F)$
      1. Q is a finte set of states
      2. Σ is a finite alphabet
      3. $δ: Q×Σ → Q$ is the transition function
      4. $q0 ∈ Q$ is the inital state
      5. $F ⊂ Q$ is the set of accept states
    3. L(M) ≡ language of all accepted strings
  7. Closure properties/Regular Operations on languages
    1. A1 and A2 are regular
    2. Union: A1 ∪ A2 = A3
    3. Concatenate: A1 A2 = A3
    4. Star: A1* = A3

2.2 Non-Deterministic Finite Automotas (NFA)

  1. Formal Definition
    1. M = (Q,Σ,δ,q0,F)
      1. Q = finite set of states
      2. Σ is a finate alphabet
      3. $δ = Q × Σε → Pow(Q)$
        1. $Σε = Σ ∪ {ε}$
      4. $q0$ = start state
      5. $F ⊂ Q$
    2. NFA accepts w If we can write $w = {y1 y2 … yn yi ∈ Σε}$ s.t. there exists a sequence of states path $R = {r0,r1,…}$
      1. r0 = q0
      2. $ri+1∈δ(ri,yi+1)$ for $i=0,1…m-1$
      3. $rm ∈ F$
  2. Useful Lemma: For all NFA, M, there exists an DFA ,N, s.t. L(m) = L(n)

2.3 Regular Expressions

  1. Def: R is a regex over a fixed alphabet iff one of the following is true:
    1. R = a ∈ Σ
    2. R = σ
    3. $R = ∅$
    4. $R = R1 ∪ R2,$ given $R1 ∧ R2$ are regex
    5. $R = R1 R2,$ given $R1 ∧ R2$ are regex
    6. $R = R*$
  2. Order of operations
    1. star
    2. concatination
    3. union
  3. Identities
    1. $a ∅ = ∅$
    2. $a σ = σ$
    3. $∅* = {σ}$

2.4 Generalized NFA (GNFA)

  1. Definition
    1. Q = set of all states
    2. $Q0 = Q - {qstart, qaccept}$
    3. The start state has out edges to every $q∈ Q - {qstart}$, and no in edges
    4. The accept state has inedges from every $q∈ Q - {qaccept}$, and no outedges
    5. An edege exists from every $q1∈ Q0$ to every $q2∈ Q0$ even if $q1=q2$
    6. Every edge is labed with a regex
  2. Useful lemma: Any NFA can be written as a GNFA
  3. lemma: Given a GNFA, M, with 2 states, the regex between the 2 states describes the language of M

2.5 Pumping Lemma for regular languages

If A is regular, then $∃ p ∈ \mathbb{N}$ s.t. $∀ s ∈ A$ for which $|s| >= p, s$ can be written as $xyz$ and satisfy the following conidition:

  1. $∀ i >= 0, x yi z ∈ A$
  2. $|y|>0$ i.e. $y ≠ ε$
  3. $|xy| <= p$

P is called the "pumping length

Problem: Given a DFA, M, with L(M) = A, find another DFA, M2c, s.t. L(M) = L(M2) and |Q2| is as small as possible

  • $δ: Q × Σ → Q$
    $\bar{δ(q,w)} q∈ Q, w∈ Σ*$
    $\bar{δ(q,w)} ≡$ interative call on delta for all $wi$ in w

    If $∃ w ∈ Σ*$ s.t. $[\bar{δ(p,w)} ∈ F$ and $δ{q,w} ∉ F]$ or $[\bar{δ(p,w)} ∉ F$ and $δ{q,w} ∈ F]$ then p and q are distrguishable

3 Context Free Grammars

3.1 Formal Definition:

  1. $(V,Σ, R, S)$
    1. $V =$ Finite set of variables or "non-terminals"
    2. Σ = finite set of terminals
      1. $Σ ∩ V = ∅$
      2. Convention: Variables are uppercase, symbols are lowercase
    3. R = finite set of rules or "subsitution rules" or "productions" 1.Rules: examples
      1. $A → aaBc | a$
        1. This means the for an A you can replace it with aaBc or a
      2. $A ⇒ OA1 ⇒ 00A11 ⇒ 001A011 ⇒ 001011$
    4. S is the start variable
  2. $L(G) = \{w∈ Σ* | S ⇒* w\}$
  3. Notation:
    1. Variables: A,B,C…
    2. Terminal: a,b,c,…0,1,$ ε $
    3. $U ⇒* V$ is defined as $∃$ sequence $U1..Uk$ , s.t. $U ⇒ U1 ⇒ U2 ⇒ … ⇒ Uk → V$

3.2 Chomsky Normal form (CNF)

  • All rules have the form
    • $A→ BC$, where $B,C$ cannot be $S$
    • $A→ a$
    • if $A→ ε$ then $A = S$
  • Lemma: Any CFG can be written in CNF

3.3 Deterministic Push Down Automotas

  • $M = (Q,Σ, Γ, δ, q0, F)$
    • $δ: Q × Σε × Γε → Q × Γε$

3.4 Non-Deterministic Push Down Automotas (PDA)

  • $M = (Q,Σ, Γ, δ, q0, F)$
    • $δ: Q × Σε × Γε → Pow(Q × Γε)$

3.5 Relating PDA to CFL

  • A language is context free iff $∃$ a PDA that recognizes it
    • Lemma: If A is CF, then $∃$ a PDA,$M$, s.t. $A = L(M)$
    • Lemma: $∀$ PDA, $M$, $∃$ CFL, $G$, s.t. $L(G)=L(M)$
      • Proof Idea: Make a conical PDA (while preserving acceptance) as follows
        1. 1 accept states
        2. Stack is empty when accepting
        3. Every transition either push or pops but not both

3.7.1 Theorems for Closure

Let Gi = (Vi, Σi, Ri, Si) for i=1,2 and Ai = L(Gi)
Without loss of generality, assume V1 ∩ V2 = ∅ ∧ S3

  • Theorem: If A1 and A2 are CFL's, then A1 ∪ A2 is a CFL
    Proof:

    Let Gi = (Vi, Σi, Ri, Si) for i=1,2 and Ai = L(Gi)
    Without loss of generality, assume V1 ∩ V2 = ∅ ∧ S3 ∉ V1 ∪ V2

    Construct G3 = (V1 ∪ V2 ∪ {S3}, Σ1 ∪ Σ2, R3, S3) with
    R3= R1 ∪ R2 ∪ {S3 → S1|S2}. \square

  • Theorem: If A1 and A2 are CFL's then A1 ⋅ A2 is a CFL
    Proof:

    ∉ V1 ∪ V2 Construct G3 = (V1 ∪ V2 ∪ {S3}, Σ1 ∪ Σ2, R1 ∪ R2 ∪ {S3 → S1 S2}, S3)

  • Theorem: If A1 and A2 are CFL's then A1* is a CFL
    Proof: Construct G3 = (V1 ∪ {S3}, Σ1, R1 ∪ {S2 → S1 S2 | ε } )
